# OwO (Emote Frog) 🐸✨

OwO (Emote Frog) is a community-driven meme coin on Ethereum (ERC-20) blending crypto culture, emojis, and frogs.  
Contract: `0x806a72273b961145cf5c5f040ad1fcd112b3f11a`
